# Chart Generator

## Overview

A simple Python project to generate basic charts using matplotlib.

---

## Features

* Bar chart generation
* Line chart generation

---

## Files

* `bar_chart.py` → create bar chart
* `line_chart.py` → create line chart

---

## Usage

```bash id="1n8d2v"
python bar_chart.py
python line_chart.py
```

---

## Use Case

* Data visualization
* Reporting
* Python plotting practice
